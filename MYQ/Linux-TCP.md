## TIME-WAIT Assassination(TWA) 现象
> TCP 的连接是由四元组（源 IP 地址，源端口，目的 IP 地址，目的端口）唯一决
定的。但是，存在这样一种情况，当一个 TCP 连接关闭，随后，客户端又使用相同的
IP 和端口号向服务端发起连接，即产生了和之前的连接一模一样的四元组。此时，如果
网络中还存在上一个连接遗留下来的包，就会出现各类的问题。
